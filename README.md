# No-Show-Appointments
Predicting whether a patient will show up on their scheduled appoinment based on their habits,illness and other factors
The project is basically about predicting whether a patient who scheduled an appointment would show up for their appoinment. The dataset contains a few patient's attributes from his general information like gender, age, location and his/her medical illness record. The projects works thorough finding correlaltion between these explanatory factors in predicting the outcome. The predictions were drawn using the approach of statistical analysis and data visaulization. The code is written in pyhton.
The results of this project can be used by hospitals and other medical institutions in predicting the traffic for the appointments for any given day.
